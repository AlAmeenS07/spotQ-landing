# spotQ

A modern queue management platform for restaurants.

## Features

* Live customer queue management
* Walk-in and remote queue joining
* Table assignment and floor management
* AI-powered wait time estimates
* Automated guest notifications
* Restaurant analytics dashboard
* Mobile-friendly responsive design

## Tech Stack

* HTML5
* Tailwind CSS
* JavaScript

## Project Structure

```text
.
├── index.html
├── style.css
├── script.js
└── images/
```

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Customize content, styles, and assets as needed.

## About

spotQ helps restaurants streamline customer flow by managing queues, table assignments, and guest updates from a single platform.
